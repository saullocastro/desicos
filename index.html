Installation¶

1) Downloading

The oficial releases are listed below:

  • Version 2.0.1
  • Version 1.0.0

The source code repository can be downloaded from GitHub or cloned using Git:

$ git clone https://github.com/desicos/desicos.git

You can also visit the source code page here.

  • scipy
